## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:7080](http://localhost:7080) with your browser to see the result.


## Note

Switch comments on declarations of `nextConfig` in `./next.config.mjs`.
